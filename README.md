# Ruihua Guo — Personal Site

Source for [ryanckoon.github.io](https://ryanckoon.github.io).

Pure static HTML + CSS — no build step, no framework. Deployed via GitHub Pages directly from the `main` branch.

- `index.html` — single-page profile
- `stylesheet.css` — all styles, design tokens at top
- `blog/` — blog index + posts + post template
- `images/` — profile photo + institution logos
- `pdf/` — CV and other downloads
- `favicon/` — favicon assets
- `.nojekyll` — tells GitHub Pages to skip Jekyll processing
